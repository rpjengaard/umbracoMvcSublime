umbracoMvcSublime
=================

Snippets for building Umbraco Mvc Templates


tabTriggers
===========

- uRenderDefaultTemplate : Render Default Template
- uRenderChildTemplate : Render Child Template
- uRenderPartialView : Render Partial View
- uRenderContentProperty : Render Umbraco Content Property (field)
- uRenderMacro : Render Umbraco Macro
